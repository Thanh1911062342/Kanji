# Kanji Mobile UI (React + Vite)

## Run
```bash
npm install
npm run dev
```

Open the URL shown in terminal.

## Data / SVG
- `public/kanji_db.json` is loaded at runtime.
- SVG files are expected at: `public/resources/kanji_svg/<codepoint_hex>.svg`

This zip includes `610` SVG files extracted from KanjiVG for the kanji found in `kanji_db.json`.
Missing SVGs: 0

If you add/update kanji list, just drop the corresponding SVGs into `public/resources/kanji_svg/`.
